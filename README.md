# vscode-extension-template

A template repository for developing VS Code extensions.

## Overview

Provides a ready-to-use environment for scaffolding VS Code extensions using `yo` and `generator-code`.

- Reference: [Your First Extension | VS Code API](https://code.visualstudio.com/api/get-started/your-first-extension)

## Structure

```text
.
├── package.json          # manages yo / generator-code
├── sample-extension/     # sample extension generated by generator-code
│   ├── src/
│   ├── package.json
│   └── ...
```

## Usage

### Generate a new extension

```bash
npm install
npm start   # runs yo code and launches the wizard to scaffold a new extension
```

### Run the sample extension

```bash
cd sample-extension
npm install
npm run compile
```

Open the `sample-extension/` folder in VS Code and press `F5` to launch the Extension Development Host.
Run `Hello World` from the Command Palette (`Cmd+Shift+P`) to verify it works.

### Key commands during development (within `sample-extension/`)

| Command | Description |
|---|---|
| `npm run compile` | Build TypeScript + lint |
| `npm run watch` | Watch for file changes and auto-build |
| `npm test` | Run tests |
| `npm run package` | Generate a production bundle |

## Tech Stack

- TypeScript
- esbuild (bundler)
- ESLint
- `@vscode/test-cli` / `@vscode/test-electron` (testing)
